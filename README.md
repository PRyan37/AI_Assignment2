# Graph Colouring Assignment

## Andrew Costello (22424842) and Padraig Ryan (22486174)  

This project implements a local search (hill climbing) approach to solve the graph colouring problem using Erdős–Rényi random graphs.

## Files

- `main.py` — Baseline experiment (Step 1)
- `part2.py` — Investigation of graph density (Step 2)

## Notes

- Uses NetworkX for graph generation
- Uses Matplotlib for visualisation
- Full explanation and results are provided in the accompanying report (PDF)
